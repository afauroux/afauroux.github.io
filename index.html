Small html/css/js projects

  • Chess RTC: quick and easy chess game, made out of javascript/html/css using the lib chess.js from @jhlywa, and allowing connxion between 2 players in p2p via webRTC
  • Timer: A simple timer/chrono page in html/js/css. Usefull for cooking pasta.
  • Beautiful Snake: This is the result of tests I did with html5 canvas: A simple webpage that allows play snake made with material design styling
  • Cat Paint: Another test of html5 canvas and hosted on firebase, that I did in collaboration with my sister, where you can paint with cat emojis üêà
  • Conway's game of life: made in a few lines of javascript. Click to draw cells, Cmd/Ctrl + click to draw rectangles, and press the spacebar to run.
  • Solitary: A game I played as a kid, that I recreated in html/js to have another go at trying to solve it (still didn't managed to).

Other projects

  • Nascam: A modular graphical user interface that I wrote in Java for a kinetic-Monte-Carlo simulation software developped at the university of Namur.
  • LowRes NX: is a retrogame virtual console. I developped several mini-games for this plateform with a friend of mine to understand better retro-consoles and to have fun. I was also involved in the porting of this console to the RetroArch plateform so games could be played on any consoles.
  • MagdaAndAntoineAndTheVan: A static blog about my travels around the world in a homemade van with my girlfriend. Made with a python script converting from markdown to html via a jinja templates, and hosted directly on github pages and google photos.
