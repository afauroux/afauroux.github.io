Hi, my name is Antoine Fauroux and I'm a doctor in plasma physics interested in many things including computer science and programming in general.

My main programming language is Python (Pandas, NumPy, Jupyter, Matplotlib), as I wrote many scripts, notebooks, and modules for physics simulations, data management, and data analysis during my studies, research internships, and my PhD. I would not release those script because they were only useful for my work but you can consult my publications here.

I accumulated many small projects over the years on my github page and this page is an attempt at advertising my programing skills by making those projects easily accessible for whomever might be interested. Here is a list of the ones that are more or less finished and accessible online:

Small html/css/js projects

  • Chess RTC: quick and easy chess game, made out of javascript/html/css using the lib chess.js from @jhlywa, and allowing connection between 2 players in p2p via webRTC
  • Snake: This is the result of tests I did with html5 canvas: A simple webpage that allows play snake made with material design styling
  • Cat Paint: Another test of html5 canvas and hosted on firebase, that I did in collaboration with my sister, where you can paint with cat emojis üêà
  • Conway's game of life: made in a few lines of javascript. Click to draw cells, Cmd/Ctrl + click to draw rectangles, and press the spacebar to run.
  • LowRes NX: is a retro-game virtual console. I developed several mini-games for this platform with a friend of mine to understand better retro-consoles and to have fun. I was also involved in the porting of this open-source console to RetroArch so games could be played on any consoles/systems.

Other projects

  • Nascam: A modular graphical user interface and visualisation tool that I wrote in Java, for a kinetic-Monte-Carlo simulation software developed at the university of Namur. I also coordinated the work of several other developers and help release this software to the scientific community.

  • MagdaAndAntoineAndTheVan: I traveled with my wife in a self-made van around North America (from Alaska to Mexico) for 8 months while I was working as a freelance software developer.

    I made a static blog about our adventures, with a simple Python script converting from Markdown to HTML via a Jinja templates, and hosted directly on Github, and using Google Photos to host the high quality pictures.
