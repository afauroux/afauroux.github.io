Hi, my name is Antoine Fauroux and I'm a doctor in physics interested in many things including computer science and programming in general.

My main programming language is python (pandas, numpy, jupyter, matplotlib), as I wrote many scripts for physics simulations, data management, and data analysis during my internships and PhD. I would not release those script because they were only useful for my work but you can consult my publications here.

I accumulated many small other projects over the years on my github page and this page is an attempt at advertising my programing skills by making those projects easily accessible for whomever might be interested.

Small html/css/js projects

  • Chess RTC: quick and easy chess game, made out of javascript/html/css using the lib chess.js from @jhlywa, and allowing connection between 2 players in p2p via webRTC
  • Timer: A simple timer/chrono page in html/js/css. Useful for cooking pasta.
  • Snake: This is the result of tests I did with html5 canvas: A simple webpage that allows play snake made with material design styling
  • Cat Paint: Another test of html5 canvas and hosted on firebase, that I did in collaboration with my sister, where you can paint with cat emojis üêà
  • Conway's game of life: made in a few lines of javascript. Click to draw cells, Cmd/Ctrl + click to draw rectangles, and press the spacebar to run.
  • Solitary: A game I played as a kid, that I recreated in html/js to have another go at trying to solve it (still didn't managed to).

Other projects

  • Nascam: A modular graphical user interface that I wrote in Java over the course of several months, for a kinetic-Monte-Carlo simulation software developed at the university of Namur. I also coordinated the work of several other developers and help release this software to the scientific community, and I am still as of today involved in improving the software and coordinating developpment.
  • LowRes NX: is a retro-game virtual console. I developed several mini-games for this platform with a friend of mine to understand better retro-consoles and to have fun. I was also involved in the porting of this console to the RetroArch platform so games could be played on any consoles.
  • MagdaAndAntoineAndTheVan: A static blog about my travels around the world in a homemade van with my girlfriend. Made with a python script converting from markdown to html via a jinja templates, and hosted directly on github pages and google photos.
